# Inferno Helper (Microbot)

Microbot port of the Inferno helper overlay. Shows safespots, upcoming attacks, prayer hints, and Zuk shield timing while running the Inferno.

## Features
- Prayer helper: tab or bottom-right icons with descending tick boxes and correct prayer highlights.
- Safespots: tile/area overlays, NPC main-tile indicators, blob detection tick, and blob death tiles.
- Wave info: current/next wave compositions with customizable colors.
- Zuk support: shield live/predict safespots plus a spawn timer infobox.
- Utility: central nibbler callout, melee dig timer, and optional non-priority box rendering.

## Usage
1. Configure:
   - **Prayer**: display mode, indicate when correct, descending boxes.
   - **Safespots**: tile vs area; enable per-NPC safespots/indicators.
   - **Waves**: show current/next wave; set text/header colors.
   - **Zuk**: choose shield display mode (live/predict) and enable the spawn timer infobox.
   - Optional: blob death tiles and melee dig timer for extra callouts.

## Requirements
- Microbot client 2.0.13+.
